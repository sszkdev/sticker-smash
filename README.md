# StickerSmash

これは[Expo公式チュートリアル](https://docs.expo.dev/tutorial/introduction/)のStickerSmashアプリです。

## 概要

StickerSmashは、ユーザーが写真を選択してステッカーを追加できるモバイルアプリケーションです。このプロジェクトはExpoとReact Nativeを使用した開発の基本を学ぶためのチュートリアルプロジェクトです。

## 機能

- 写真の選択と表示
- ステッカーの追加と配置
- 完成した画像の保存と共有

## 技術スタック

- Expo
- React Native
- TypeScript
- Expo Router

## 開始方法

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npx expo start
```
